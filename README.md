# AI Call Frontend

Минимальный frontend для тестирования real-time перевода голосовых разговоров между двумя пользователями.

## Особенности

- Генерация или ввод ID сессии
- Кнопка «Создать / Подключиться»
- Выбор языков RU ↔ EN
- Статус соединения
- Подключение к backend на Render через WSS

## Настройка подключения к серверу

Frontend подключается к WebSocket серверу через конфигурационный файл `config.js`.
По умолчанию используется URL: `ai-call-backend-esj7.onrender.com` (реальный URL сервера).

## Важное замечание о доступности

Сайт https://wondrous-jalebi-bde9b9.netlify.app/ может быть недоступен из-за достижения лимитов Netlify.
Для тестирования приложения рекомендуется развернуть frontend на собственном аккаунте Netlify.

Если вам нужно изменить URL сервера:
1. Отредактируйте файл `config.js`
2. Замените `window.BACKEND_WS_URL` на ваш URL сервера

## Деплой на Netlify

1. Перейдите на https://app.netlify.com/drop
2. Загрузите все файлы репозитория
3. Frontend подключается к backend на Render через WSS
